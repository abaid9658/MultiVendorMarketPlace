VendorPro - Vendor Management & Quotation System
===================================================
Task ID: FS-2 | Full-Stack Web Development | TEYZEX CORE Internship

Short Description:
------------------
A full-stack web-based Procurement and Vendor Management system. Enables organizations to manage vendor directories, create RFQ/quotation requests, log activities in real-time, and compare bidding quotes side-by-side to highlight the most cost-effective deal.

Technologies Used:
------------------
- React.js, Vite, Tailwind CSS, jsPDF
- Node.js, Express.js, Socket.io
- MongoDB Atlas, Mongoose

Instructions to Run:
--------------------
1. Setup .env file inside 'server/' folder (see server/.env.example)
2. Open terminal in the 'server/' directory, run:
   - npm install
   - npm run seed (to populate default admin accounts & sample data)
   - npm run dev (to start backend on port 5000)
3. Open terminal in the 'client/' directory, run:
   - npm install
   - npm run dev (to start client on port 5173)
